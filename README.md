# 🤖 Face Mask Detection System 😷

## 📌 About the Project

This project is a real-time face mask detection system that uses computer vision and deep learning. It detects whether a person is wearing a mask or not using a webcam and gives voice feedback accordingly.

I built this project as part of my robotics work, where the system can also be integrated with a robot (EZ-Robot using ARC) to respond in real-time.

## 🚀 Features

* Detects faces using OpenCV
* Classifies Mask / No Mask using a trained CNN model
* Provides voice feedback using Python
* Works in real-time using webcam

## 🛠️ Technologies Used

* Python
* OpenCV
* TensorFlow / Keras
* NumPy
* pyttsx3

## ▶️ How to Run

1. Install required libraries:
   pip install opencv-python tensorflow numpy pyttsx3

2. Run the main file:
   py -3.10 mask_detect.py

## 📷 Output

* Face is detected from webcam
* Label is displayed (Mask / No Mask)
* Voice output is generated based on result
